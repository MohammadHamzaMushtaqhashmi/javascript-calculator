# 📖 JavaScript Calculator <a name="about-project"></a>

**JavaScript Calculator** is a web application that performs basic arithmetic operations. Built with React and Bootstrap, it is designed to provide a user-friendly interface for calculations.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Node.js
- npm

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/yourusername/javascript-calculator.git
```

### Install

Install this project with:

```sh
  cd javascript-calculator
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```

### Deployment

You can deploy this project using:

```sh
  npm run build
  npm run deploy
```
